    # 自我补充linux操作：

    touch 文件名 创建文件
    mkdir 文件夹 创建文件夹
    cd 文件夹 进入文件夹
    ll/ls 查看文件夹里的文件
    ls -l 查看文件的属性
    vi/vim 文件名 编辑文件 退出：按ESC，输入:q!回车
    cat/less 文件名 查看文件
    rm [-dfirv][--help][--version][文件或目录...] 删除目录，文件
    执行rm指令可删除文件或目录，如欲删除目录必须加上参数”-r”，否则预设仅会删除文件

    # 补充获取到ssh key

    -bash: /root/.ssh: Is a directory; /root/.ssh是.ssh文件夹文件夹路径

    进入文件夹

    cd /root/.ssh

    查看id_rsa.pub文件

    cat id_rsa.pub

    文件内容

    ssh-rsa AAAAB3NzaC1yc2EAAAADA...此处省略...E3 2929166164@qq.com
