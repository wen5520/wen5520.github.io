    # jenkins集成coding

    jenkins 集成coding实现自动化部署

    Jenkins的凭证管理

    致流程如下：

    1、提交代码至coding 2、coding收到合并请求或者push命令 则触发向jenkins 发送数据请求 3、当jenkins 收到数据触发构建器 且执行构建步骤执行 4、构建步骤内容： a、连接云服务器 b、推送代码至指定服务器目录 c、执行 commend (可以是任何脚本命令)
